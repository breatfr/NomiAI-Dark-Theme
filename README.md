# NomiAI dark theme
Stylus dark theme for chat pages of [NomiAI](https://beta.nomi.ai) website

## How to use in few steps
1째) Install Stylus browser extension
- Chromium browsers link: https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne
- Firefox link: https://addons.mozilla.org/firefox/addon/styl-us/
  
2째) Go on [NomiAI](https://beta.nomi.ai) website and click on Stylus icon in your browser and click on the URL under "Write a style for" to create a new style

3째) Click on Import button and copy all the code from [nomi.ai-dark-theme.css](https://github.com/breatfr/NomiAI-dark-theme/blob/main/nomi.ai-dark-theme.css) to your new Stylus style and save.

4째) Enjoy :)
